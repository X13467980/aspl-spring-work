## 2. 実験内容

### 2.1 TSP信号の実装（tsp_gen）

**概要:** Time Stretched Pulse（時間伸長パルス）信号を生成。周波数領域で位相を設計し、IFFT で時間領域に変換。

**パラメータ:**
- 信号長: N = 262144 サンプル（2^18）
- 実行長: J = N/2
- サンプリング周波数: 48 kHz
- 出力: `tsp_signal.wav`（約 5.46 秒）

**原理:** 周波数領域で位相 θ(k) = 2πJ(k/N)² を設計し、IFFT で時間伸長パルスを生成。再生後、逆フィルタ（down-TSP）を適用することでインパルス応答が得られる。

---

### 2.2 白色信号の実装（white_noise）

**概要:** 一様乱数を用いたホワイトノイズを生成。

**パラメータ:**
- サンプリング周波数: 48 kHz
- デフォルト: 180 秒
- 秒数・出力ファイル名をコマンドラインで指定可能

**使用例:**
```bash
./white_noise                    # 180秒 → white_noise_180s.wav
./white_noise 60 my_noise.wav    # 60秒 → my_noise.wav
```

---

### 2.3 TSP応答からインパルス応答の算出（tsp_to_ir）

**概要:** TSP信号とその応答から、周波数領域で逆フィルタ（down-TSP）を適用してインパルス応答を抽出。

**手順:**
1. TSP信号 S(k) と応答 Y(k) を FFT
2. 逆フィルタ INV(k) = exp(+j·2πJ·(k/N)²) を計算
3. H(k) = Y(k) · INV(k) / S(k) で周波数領域のインパルス応答を算出
4. IFFT で時間領域に変換

**ノイズ低減:** 複数の応答ファイルを指定すると、時間領域で平均してから IR を算出。

**使用例:**
```bash
./tsp_to_ir recordings/tsp_signal.wav impulse_response_tsp.wav recordings/tsp_1.wav
make tsp_to_ir_all   # tsp_1～10 を平均
```

---

### 2.4 白色信号＋適応フィルタでインパルス応答の算出（adaptive_filter）

**概要:** 白色信号とその応答から、NLMS（Normalized Least Mean Squares）適応フィルタでインパルス応答を推定。

**アルゴリズム（NLMS）:**
- ステップサイズ: μ = 0.1
- 正則化: ε = 1e-6
- 更新式: w[n+1] = w[n] + (μ / (||x||² + ε)) · e · x
- 誤差: e = d - y_hat（応答 - 推定出力）

**パラメータ:**
- フィルタ長: デフォルト 48000 サンプル（1秒）
- 入力: 白色信号 180 秒、応答録音

**使用例:**
```bash
./adaptive_filter white_noise_180s.wav recordings/white_noise_recorded.wav impulse_response_white.wav 48000
```

---

### 2.5 残響時間の分析（ir_analyze）

**概要:** インパルス応答から Schroeder 積分により残響曲線を計算し、T10・T20・RT60 を算出。

**手順:**
1. **残響曲線:** インパルス応答の2乗の後方累積積分 E(t) = ∫[t→∞] h²(τ)dτ
2. **T10:** 残響曲線の -5 dB ～ -15 dB の区間で線形回帰
3. **T20:** 残響曲線の -5 dB ～ -25 dB の区間で線形回帰
4. **RT60:** T10×6、T20×3 で 60 dB 減衰時間を近似
5. **ノイズ対策:** ピークから -40 dB 以下で打ち切り

**使用例:**
```bash
./ir_analyze impulse_response_tsp.wav
./ir_analyze impulse_response_white.wav decay_curve.txt
make plot_decay   # gnuplot で decay_curve.png を生成
```

---

### 2.6 逆フィルタ（ir_to_inverse）【発展課題】

**概要:** インパルス応答から正則化付き逆フィルタを算出。空間伝達に畳み込むと理想的なインパルスに近づく（補償）。

**式:** H_inv(k) = conj(H(k)) / (|H(k)|² + ε)

**使用例:**
```bash
./ir_to_inverse impulse_response_tsp.wav inverse_filter_tsp.wav
./ir_to_inverse impulse_response_white.wav inverse_filter_white.wav
```

### 3.5 二つの応答の比較

| 項目 | TSP | 白色＋適応フィルタ |
|------|-----|-------------------|
| インパルス応答長 | 5.46 秒 | 1.00 秒 |
| T10 | 0.012 秒 | 0.012 秒 |
| T20 | 0.024 秒 | 0.025 秒 |
| RT60 (T10から) | 0.070 秒 | 0.072 秒 |
| RT60 (T20から) | 0.071 秒 | 0.075 秒 |

**考察:**
- 両手法とも同程度の残響時間（RT60 約 0.07 秒）が得られた
- TSP と白色＋適応フィルタの結果は一致しており、同一空間の伝達特性を正しく計測できている
- 収録場所は残響時間が短い条件（2/25 のタスク）に適合
