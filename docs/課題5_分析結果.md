# 課題5: 空間伝達特性の分析結果

3（TSP）と4（白色信号＋適応フィルタ）で計測したインパルス応答を、ir_analyze で解析した結果。

## 分析条件

- サンプリング周波数: 48 kHz
- 残響曲線: Schroeder積分（インパルス応答の2乗積分）
- T10: -5 dB ～ -15 dB の区間で線形回帰
- T20: -5 dB ～ -25 dB の区間で線形回帰
- RT60: T10×6、T20×3 で60 dB減衰時間を近似
- ノイズ対策: ピークから -40 dB 以下で打ち切り

## 結果比較

| 項目 | TSP | 白色＋適応フィルタ |
|------|-----|-------------------|
| インパルス応答長 | 5.46 秒 (262144 サンプル) | 1.00 秒 (48000 サンプル) |
| 有効長（ノイズ打ち切り後） | 4.22 秒 | 0.13 秒 |
| **T10** | 0.012 秒 | 0.012 秒 |
| **RT60 (T10から)** | 0.070 秒 | 0.072 秒 |
| **T20** | 0.024 秒 | 0.025 秒 |
| **RT60 (T20から)** | 0.071 秒 | 0.075 秒 |

## 考察

- 両手法とも同程度の残響時間（RT60 約 0.07 秒）が得られた
- TSP と白色＋適応フィルタの結果は一致しており、同一空間の伝達特性を正しく計測できている
- 収録場所は残響時間が短い条件（2/25 のタスク）を想定

## 発展課題: 逆フィルタ（ir_to_inverse）

3と4のインパルス応答から逆フィルタを算出：

```bash
./ir_to_inverse impulse_response_tsp.wav inverse_filter_tsp.wav
./ir_to_inverse impulse_response_white.wav inverse_filter_white.wav
```

- 正則化付き逆フィルタ: H_inv = conj(H) / (|H|² + ε)
- 逆フィルタを空間伝達に畳み込むと、理想的なインパルスに近づく（補償）
