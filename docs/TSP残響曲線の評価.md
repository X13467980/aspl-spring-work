# TSP 残響曲線の評価

## 結果サマリ

| 項目 | 値 |
|------|-----|
| T10 | 0.012 秒 |
| T20 | 0.024 秒 |
| RT60 (T10から) | 0.070 秒 |
| RT60 (T20から) | 0.071 秒 |
| T10/T20 区間 | 4.197 ～ 4.217 秒 |

## 理想との比較

### 理想的な残響曲線の条件

1. **直線性**: 拡散音場では、Schroeder 積分の残響曲線は dB 軸でほぼ直線的に減衰する
2. **T10 と T20 の整合性**: T20 ≈ 2×T10（直線減衰なら成立）
3. **RT60 の整合性**: T10 からと T20 から求めた RT60 が近い値になる

### 今回の TSP 結果の評価

| 観点 | 結果 | 評価 |
|------|------|------|
| T20 / T10 | 0.024 / 0.012 = 2.0 | ✓ 整合 |
| RT60 (T10) vs RT60 (T20) | 0.070 vs 0.071 秒 | ✓ ほぼ一致 |
| 減衰の直線性 | 4.196～4.197 秒で急落、4.198～4.199 秒で一時的に平坦 | △ やや不規則 |

### 考察

- **良い点**
  - T10 と T20 から求めた RT60 が一致しており、線形回帰の区間選びは妥当
  - 残響時間 0.07 秒は「残響時間が短い」タスク（2/25）に合致

- **理想から外れている点**
  - 残響曲線に平坦な区間（約 -9.4 dB で 10 ms 程度）があり、完全な直線減衰ではない
  - 離散的な反射や室モードの影響が考えられる
  - 小さい部屋では拡散音場の仮定が成り立ちにくく、この程度の揺らぎはあり得る

### 結論

**「理想」かどうか**: 完全な理想形ではないが、実測としては妥当な結果。

- 数値的には T10/T20/RT60 の整合性が取れている
- 曲線の形状は、小さい部屋や離散反射の影響で理想的な直線からややずれることがある
- タスク（残響時間が短い条件）を満たしており、計測としては問題ない
